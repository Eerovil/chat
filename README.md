# Papo-Reto
Real time chat built with Flask and Websockets. Available at [Papo-Reto](https://papo-io.herokuapp.com).

## Installing

### Cloning
```bash
$ git clone https://github.com/raymag/papo-reto.git
$ cd papo-reto
$ pip install -r requirements.txt
```

### Running
```bash
$ python main.py
```
